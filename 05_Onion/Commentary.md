# 問題について
何らかのアルゴリズムで暗号化されているデータです。  
何らかの暗号といえば。 

 - ハッシュ
 - Base64
 - RSA  

様々ですが、よく見ると解ると思います。  



# 解法
1.	問題タイトルの時点である程度ヒントがでています。  

1.  気が付いた方はヒントを見なくても大丈夫ですね。
	
	- 暗号文に記号が含まれていないため、base64のエンコードである。　

	- この文章は多重にエンコードされたBase64である。  

	Onionとは、玉ねぎの皮のように何重にも重なった処理を意味しています。




1.	デコードしましょう。先程の文を保存したファイルをChiperとします。
	```
	nyancat86@ubuntu:~/git/ksnctf/05_Onion/work$ ls
	Chiper
	```



1.	```cat Chiper | base64 -d```  とすると一段目のデコードをかけることができます。

	```
	nyancat86@ubuntu:~/git/ksnctf/05_Onion/work$ cat Chiper | base64 -d 
	```
